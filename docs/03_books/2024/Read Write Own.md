# Memo

"한마디로 블록체인 네트워크는 더 나은 인터넷을 만들기 위한 새로운 건축자재다." - p.29

기업 네트워크는 언제나 '유인-추출 사이클(attract-extract cycle)' 논리를 따른다. 협력에서 경쟁으로의 전이는 보완재에 대한 배신처럼 느껴진다. 시간이 지날수록 최고의 기업가, 개발자, 투자자는 기업 네트워크를 기반으로 무언가 만드는 것을 경계하게 된다. - p.95
